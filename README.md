# BYA MarketDesk

MVP сервиса в духе CryptoCompare в стилистике BYAcademy: актуальные котировки, рыночная таблица, watchlist и личный портфель с расчетом стоимости и P/L.

## Что уже есть

- Live-курсы через CoinGecko `/coins/markets`.
- Топ-80 монет по капитализации.
- Переключение валюты: USD, EUR, RUB, GBP.
- Поиск по названию и тикеру.
- Сортировка по рангу, изменению за 24 часа и объему.
- Watchlist с сохранением в `localStorage`.
- Личный портфель с объединением повторных покупок одной монеты, средней ценой входа, текущей стоимостью и P/L.
- Локальная регистрация профиля с именем, email и ролью, подготовленная под будущую почтовую авторизацию.
- Разбивка портфеля по долям, лучший/худший актив и вклад за 24 часа.
- Fear & Greed Index через Alternative.me.
- BTC dominance, число активных рынков и глобальные метрики CoinGecko.
- Живая BTC-цена, глобальное изменение рынка и автообновление данных без перезагрузки страницы.
- Realtime-поток цен через Binance WebSocket для ликвидных USDT-пар из топа рынка, портфеля и watchlist.
- Индикатор альтсезона: доля топ-альтов, которые обгоняют BTC по 30-дневной динамике.
- Investor ROI / VC Portfolio: DropsTab-style сигнал по венчурной поддержке, ROI, качеству фондов и ликвидности.
- Расширенная таблица: 1ч, 24ч, 7д, 30д, расстояние до ATH и рост от ATL.
- Лидеры и аутсайдеры движения рынка.
- Экспорт портфеля в CSV.
- Fallback-данные, если публичный API временно недоступен.

## Запуск

Из папки проекта:

```powershell
py -3 -m http.server 5187 --bind 127.0.0.1
```

Открыть:

```text
http://127.0.0.1:5187/
```

Для живой RSS-ленты Investing.com лучше запускать через встроенный Node proxy:

```powershell
node server.mjs
```

Открыть:

```text
http://127.0.0.1:5188/
```
